Integration points

  • Members.affluent.co academy page: VSL embed + waitlist capture form. No other change.
  • affluent.academy router: Untouched. Existing revenue-tier router stays as the call-funnel entry. Scorecard sits one step upstream as the warm-list capture before the call-funnel decision.
  • Existing email tool: New segments created. Existing flows untouched.
  • Existing Stripe: New product (cohort enrolment) added. Existing Academy and Agency Launch products untouched.
  • Existing CRM: New tags created for scorecard segments + cohort applicants. Existing tagging untouched.

What this build does NOT touch

  • Academy curriculum
  • Agency Launch sales process
  • Agency Launch guarantee structure
  • Affluent.co DTC agency funnel
  • Appointwise / Clay / Skool / TripleWhale software investments
  • Existing podcast or YouTube cadence
  • Existing book funnel (15minuteagency.com)

The three new surfaces slot between the existing surfaces. They do not replace anything.
